## Overview

Welcome to our Project Pitch Directory! This platform is designed for creators and innovators to showcase their projects and gather attention from peers, contributors, or potential partners. Whether you're building an open-source tool, a startup MVP, or an experimental side project, you can share your work here.

Once authenticated with **GitHub OAuth**, you can submit your pitch, complete with detailed markdown descriptions, images, and project metadata.

## Key Features

- **Project Submission with Markdown**: Submit detailed project descriptions using our built-in **Markdown Editor (mdeditor)**, making it easy to format your pitch.
- **Authentication via GitHub**: Securely sign in and manage your pitches using **GitHub OAuth**.
- **Slugify for SEO-friendly URLs**: Each project automatically gets a clean and readable URL slug based on the project title.
- **Validation with Zod**: Ensure project submissions follow a structured and validated schema using **Zod**.
- **UI Powered by shadcn/ui**: The platform features a modern and accessible UI library for a seamless user experience.
- **Error Monitoring with Sentry**: Any issues during submission or browsing are captured and monitored via **Sentry** for proactive improvements.

## How it Works

1. **Sign in**: Use your GitHub account to log in securely.
2. **Submit a Pitch**: Fill out project details including title, description, images, and team information.
3. **Showcase**: Your project will be listed in our public directory for others to explore.
4. **Edit Anytime**: You can update your project info later as your project evolves.

## Technologies Behind the Platform

- **Next.js**: Framework for the core frontend and backend logic.
- **shadcn/ui**: For building clean and reusable UI components.
- **mdeditor**: Markdown-based content editor for writing rich project pitches.
- **Zod**: Schema validation for user-submitted data.
- **Slugify**: For generating SEO-friendly URLs based on project titles.
- **Sentry**: Error and performance monitoring.
- **GitHub OAuth**: Secure authentication for users.

## Why Pitch Here?

We believe in amplifying creative and technical projects by giving them a home where people can explore, collaborate, and provide feedback. Whether you are looking for contributors, showcasing to potential users, or building your portfolio, this directory is designed to help you pitch confidently.
